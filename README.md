 Tic-Tac-Toe (Jogo da Velha):

Este projeto é uma implementação simples e funcional do clássico Tic-Tac-Toe (Jogo da Velha) utilizando HTML, CSS e JavaScript puro.
O objetivo é oferecer uma interface limpa, responsiva e intuitiva, permitindo que dois jogadores disputem partidas diretamente no navegador.

Como funciona:

O tabuleiro é composto por 9 células, organizadas em uma grade 3×3.

Dois jogadores se alternam clicando nas células para jogar como X e O.

O sistema verifica automaticamente:

Linhas

Colunas

Diagonais

Quando uma combinação vencedora é detectada, o jogo exibe o vencedor e desenha uma linha sobre a jogada vencedora.

Se todas as posições forem preenchidas sem vencedor, o jogo declara “Deu Velha”.

Recursos adicionais:

Botão de Reiniciar, que limpa o tabuleiro e recomeça a partida.

Layout totalmente responsivo, funcionando bem em computadores e dispositivos móveis.

Código organizado para facilitar manutenção e expansão (ex.: adicionar bot vs player futuramente).

Tecnologias utilizadas:

HTML5 — estrutura do tabuleiro e botões.

CSS3 — estilização e responsividade do layout.

JavaScript — lógica do jogo, alternância de turnos, verificação do vencedor e reinício da partida.
